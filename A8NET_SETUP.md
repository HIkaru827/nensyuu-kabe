# A8.net 広告の設定方法

## 概要
このプロジェクトでは、A8.netのアフィリエイト広告を簡単に設置できるように設定されています。
環境変数を使って広告リンクを管理するため、後から変更も簡単です。

## 設定手順

### 1. A8.netで広告主と提携する

以下のバイト求人サービスと提携することをおすすめします：

- **タウンワーク**（リクルート）- 最も知名度が高い
- **マッハバイト**（旧ジョブセンス）- 祝い金がもらえるため学生に人気
- **バイトル**（ディップ）- 動画で職場の雰囲気がわかる
- **アルバイトEX**（じげん）- 複数サイトを一括検索
- **ショットワークス** - 短期・単発バイト向け

### 2. 広告リンクを取得する

A8.netの管理画面で：
1. 「プログラム管理」→ 「参加中プログラム」を開く
2. 提携した広告主の「広告リンク」をクリック
3. テキストリンクまたはバナーリンクをコピー

**注意**: 広告リンクの形式は `https://px.a8.net/svt/ejp?...` のようなURLです。

### 3. `.env.local`ファイルに追加する

プロジェクトのルートディレクトリにある `.env.local` ファイルに以下を追加してください：

```bash
# Google Analytics（既存）
NEXT_PUBLIC_GA_ID=G-FKSP61MD9N

# A8.net バイト求人広告リンク
# 各サービスのアフィリエイトリンクをA8.netから取得して貼り付けてください

# タウンワーク（リクルート）
NEXT_PUBLIC_A8_TOWNWORK="https://px.a8.net/svt/ejp?..."

# マッハバイト（旧ジョブセンス）
NEXT_PUBLIC_A8_MACHBAITO="https://px.a8.net/svt/ejp?..."

# バイトル（ディップ）
NEXT_PUBLIC_A8_BAITORU="https://px.a8.net/svt/ejp?..."

# アルバイトEX
NEXT_PUBLIC_A8_ARBEIT_EX="https://px.a8.net/svt/ejp?..."

# シミュレーション結果に応じた広告リンク（オプション）
# 高時給バイト向け（年収160万円を目指す場合）
NEXT_PUBLIC_A8_HIGH_WAGE="https://px.a8.net/svt/ejp?..."

# シフト調整可能なバイト向け（年収130万円以下に抑える場合）
NEXT_PUBLIC_A8_FLEXIBLE="https://px.a8.net/svt/ejp?..."

# おすすめバイト（年収130万円以下の場合）
NEXT_PUBLIC_A8_RECOMMENDED="https://px.a8.net/svt/ejp?..."

# キャリアアップ求人（年収160万円以上の場合）
NEXT_PUBLIC_A8_CAREER="https://px.a8.net/svt/ejp?..."

# バナー広告（オプション）- HTMLコード全体を1行で記述
NEXT_PUBLIC_A8_BANNER="<a href='https://px.a8.net/...'><img src='...' /></a>"
```

### 4. 開発サーバーを再起動する

環境変数を変更した後は、開発サーバーを再起動してください：

```bash
# Ctrl+C でサーバーを停止
pnpm dev
```

## 広告が表示される場所

### 1. バイト求人広告（JobAdSlot）
- **場所**: シミュレーション結果の下
- **表示**: 最大4つのバイト求人サービスが2x2のグリッドで表示
- **条件**: 環境変数にURLが設定されているもののみ表示

### 2. アクションボタン
- **場所**: シミュレーション結果の直後
- **表示**: 年収に応じて異なるメッセージとリンク
  - 年収130万円未満: 「おすすめバイト」
  - 年収130-160万円: 「高時給バイト」「シフト調整可能なバイト」
  - 年収160万円以上: 「キャリアアップ求人」

### 3. バナー広告（オプション）
- **場所**: バイト求人広告の下
- **表示**: A8.netから取得したバナー広告HTML

## カスタマイズ

### 広告の表示/非表示を切り替える

`.env.local`でURLを設定しなければ、その広告は表示されません。

例：タウンワークだけ表示したい場合
```bash
NEXT_PUBLIC_A8_TOWNWORK="https://px.a8.net/svt/ejp?..."
# 他は設定しない
```

### 広告の見た目を変更する

`components/ad-slot.tsx`を編集してください。

### 広告の配置場所を変更する

`components/income-simulator.tsx`を編集してください。

## トラブルシューティング

### Q: 広告が表示されない
A: 以下を確認してください：
1. `.env.local`にURLが正しく設定されているか
2. URLが `"` で囲まれているか
3. 開発サーバーを再起動したか

### Q: リンクが「#」になっている
A: 環境変数が設定されていません。`.env.local`を確認してください。

### Q: バナー広告が正しく表示されない
A: HTMLコードを1行にして、シングルクォートとダブルクォートのエスケープに注意してください。

```bash
# ⭕ 正しい例
NEXT_PUBLIC_A8_BANNER="<a href='https://...'><img src='...' /></a>"

# ❌ 間違った例（改行がある）
NEXT_PUBLIC_A8_BANNER="<a href='https://...'>
<img src='...' />
</a>"
```

## 収益を最大化するコツ

1. **複数のサービスと提携する**: ユーザーの好みに合わせて選べるように
2. **特徴を明記する**: 「祝い金」「高時給」などのタグで差別化
3. **適切な場所に配置**: シミュレーション結果の直後が最もクリック率が高い
4. **定期的に見直す**: 成果を確認して、パフォーマンスの低い広告は入れ替える
